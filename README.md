# Restaurant Management System

## Overview
A C++ based restaurant management system designed to manage customers, employees, menu items, orders, and payment processing.

## Technologies Used
- C++
- Object-Oriented Programming
- STL (Vector, Smart Pointers, Unordered Map)

## Concepts Implemented
- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Composition and Aggregation

## Features
- Admin login system
- Menu item management
- Customer order creation
- Table booking system
- Cash and card payment handling
- Order display and management
